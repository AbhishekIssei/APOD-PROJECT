### 1. Clone the Repository


### 2. Get a NASA API Key

* Go to [https://api.nasa.gov/](https://api.nasa.gov/)

### 3. Add Your API Key

Open the `script.js` file and replace the `DEMO_KEY` with your own API key:

